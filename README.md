# Credit Card Fraud Detection - Live Dashboard

## Background
Real-time fraud detection dashboard built with Streamlit, 
simulating live transaction monitoring system similar to 
payment switching infrastructure. Supported by my experience 
as IT Monitoring staff at a payment switching company.

## Features
- Real-time transaction feed with RRN, Card Number, Account, Merchant
- Fraud alert notification
- Time range filter (15 minutes, 1 hour, Today, 2 days, Custom)
- Search by RRN and Card Number
- Filter by transaction status (Normal/Fraud)
- Real-time pie chart and bar chart analysis

## Live Demo
https://fraud-detection-dashboard-9pspmc29agpf6almnso82u.streamlit.app/

## Dataset
- Source: Kaggle - Credit Card Fraud Detection (ULB)
- Total transactions: 284,807
- Fraud transactions: 492 (0.17%)
